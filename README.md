# 💰 Fin User

Sistema web de controle financeiro pessoal desenvolvido como projeto acadêmico, com foco em organização financeira, experiência do usuário e visual moderno estilo fintech.

O projeto foi inicialmente planejado através de prototipação UI/UX no Figma e futuramente será desenvolvido utilizando tecnologias web modernas.

---

# 🎯 Objetivo do Projeto

O Fin User tem como objetivo ajudar usuários a:

- Controlar receitas e despesas
- Visualizar sua saúde financeira
- Organizar gastos mensais
- Definir metas financeiras
- Receber insights financeiros
- Melhorar sua organização financeira de forma simples e intuitiva
