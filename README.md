# 💰 Fin User

Projeto acadêmico de um sistema de controle financeiro pessoal.

Este repositório contém a fase inicial do projeto, focada em **protótipo de interface (UI/UX)** desenvolvido no Figma.

---

## 🎯 Objetivo

Criar uma aplicação que ajude usuários a controlar suas finanças pessoais de forma simples, visual e intuitiva.
